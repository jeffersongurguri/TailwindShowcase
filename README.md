Bem-vindo ao repositório de exemplos práticos de interface utilizando **Tailwind CSS** (v4), desenvolvido para as aulas de Front-end do curso de **Informática para Internet** do **IFCE - Campus Baturité**.

Este repositório contém exemplos focados em prototipagem rápida, utilizando a versão de navegador (CDN) do Tailwind, o que permite testar ideias e validar layouts sem a necessidade de configurar um ambiente de *build*.

---

## 📂 O que você vai encontrar aqui

O repositório é composto por 3 exemplos de componentes de interface, aplicados a contextos específicos:

1. **Card de Divulgação de Curso:** Demonstração de layout estático com Flexbox, arredondamentos (border-radius), sombras e efeitos de *hover* aninhados para interação visual (utilizando a classe `group`).
2. **Painel de Status de Projeto (Laboratório Maker):** Aplicação prática de CSS Grid, Flexbox avançado para alinhamento e criação de indicadores visuais animados (pontos de status de progresso).
3. **Formulário de Acesso:** Estilização de *inputs* e botões, com forte foco em acessibilidade e *feedback* visual para o usuário através dos estados de foco (`focus:` e `ring`).

---
